## 调试和构建脚本 Debug & Build Scripts

```powershell
## 本地调试脚本 CMD

## Script Path: applicatons/web-application/package.json

### DEBUG
pnpm run debug

### DAILY-BUILD BUILD
pnpm run build:daily-build

### BETA BUILD
pnpm run build:beta

### PRE-RELEASE BUILD
pnpm run build:pre-release

### RELEASE BUILD
pnpm run build:release
```
